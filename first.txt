コミットは1作業ごとに行う。
コミットする変更内容はステージングエリアで選択する。
コメントは端的に分かりやすく書く。


ls
ls -a

git init
    ローカルリポジトリ作成
git add xxxxxxx.yyy
    変更をステージングエリアへ登録する
git add .
    ステージングエリアへ移動
git rm xxxxxxx.yyy
    ステージングエリアからファイルを削除
git commit -m "コメントを書く"
    ローカルリポジトリへコミット
git commit -v
    変更内容を確認してコミット
git status
    リポジトリ内の状態確認
git log
    コミットログの表示

rm
    ファイルの削除
